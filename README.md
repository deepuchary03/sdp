
## Music Player (Spotify Clone)[MERN]
A simple website similar to Spotify in which you can stream songs, and upload through Admin panel.

#Technologies Used:

## Live App link : https://deepuchary.vercel.app/

## Tech Stack

**Client:** React, Redux, TailwindCSS

**Server:** Node, Express

**Database:** MongoDB
